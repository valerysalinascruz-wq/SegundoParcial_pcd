# 🎓 Sistema de Gestión de Calificaciones (versión robusta)

Sistema de gestión académica desarrollado en Python con **Pandas y NumPy**, que permite generar datos sintéticos, analizarlos y producir reportes académicos completos.

Proyecto para la materia **Programación para Ciencia de Datos – IPN (2026)**.

---

## 📋 Descripción

Este sistema simula un entorno académico con:

- 15 estudiantes  
- 7 materias  
- Calificaciones por parcial (parcial_1, parcial_2, final)  
- Valores nulos simulados en algunos registros  

A partir de estos datos, el sistema permite realizar análisis académicos completos:

- Información general del sistema  
- Validación de calidad de datos  
- Búsqueda de estudiantes  
- Generación de kardex individual  
- Cálculo de estadísticas por materia y semestre  
- Ranking de estudiantes  
- Detección de estudiantes en riesgo académico  
- Reporte académico general  
- Exportación de kardex en CSV o JSON
